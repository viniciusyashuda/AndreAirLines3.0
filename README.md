# AndreAirLines3.0

(*)• Projeto Andre AirLines com Micro Serviços ✔️ (*)
(*)• Bancos de dados utilizados: SQL Server e MongoDB ✔️ (*)
(*)• Comparação de performance de busca Entity Framework x Dapper ✔️ (*)

# Observações
